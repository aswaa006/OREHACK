## 🔧 Customization Guide

### Change Primary Color
In `AdminDashboard.css`, update:
```css
:root {
  --accent-cyan: #YOUR-COLOR; /* All cyan will update */
}
```

### Adjust Animation Speed
```css
:root {
  --transition-fast: 100ms; /* Faster animations */
  --transition-base: 200ms;
  --transition-slow: 400ms;
}
```

### Add Custom Component Variant
```jsx
// In PremiumComponents.jsx, add variant class
.button-custom { /* your styles */ }

// Use in component
<PremiumButton variant="custom">Button</PremiumButton>
```

### Change Font Family
```css
:root {
  --font-display: 'Your Font', sans-serif;
  --font-body: 'Your Font', sans-serif;
}
```

## 🚨 Common Issues & Quick Fixes

### Animations not working?
```jsx
// Make sure CSS is imported BEFORE components
import './styles/AdminDashboard.css';
import { PremiumEventCard } from './components/PremiumComponents';
```

### Colors look wrong?
Check your CSS variables are set:
```css
:root {
  --accent-cyan: #00d9ff;
  --bg-primary: #0a0e27;
}
```

### Mobile layout broken?
Ensure media queries are being applied:
```css
@media (max-width: 768px) {
  /* Mobile styles */
}
```

### Hover effects not visible?
Check backdrop-filter support and add webkit prefix:
```css
.card-glass-bg {
  backdrop-filter: blur(16px);
  -webkit-backdrop-filter: blur(16px);
}
```

### Performance issues?
Use Chrome DevTools:
1. Open DevTools > Performance
2. Record interaction
3. Look for 60fps (green) vs. janky (red)
4. Optimize based on findings
